Preview, Edit & Tools

Inspect and work with file contents

Preview

  • F3 — open an in-app preview for the focused item
  • ⌃⇧P — show or hide the bottom preview pane (follows focus)
  • ⌘Y or ⌥F3 — system Quick Look

Supported previews include images, PDF, text/source files, and common audio/video formats. Other types show metadata and a Quick Look button.

Edit

  • F4 — open the text editor for text-like files, or the hex editor for binary files
  • ⌘⌥H — open the hex editor explicitly

Terminal

Press ⌃` to show the embedded terminal. It can sync to the active panel’s folder (cd Active). Use External to open Terminal.app in that directory. The terminal and bottom preview pane are mutually exclusive.

Compare panels

⌘⌥D compares the left and right folders and lists items that exist only on one side, are identical, or differ (by name/size/date rules in the compare UI).

Find duplicates

⌘⇧D scans for files with identical contents (hash-based) and lets you review groups and trash unwanted copies.

Git status

When the active folder is inside a Git repository, changed files can show status badges, and the status bar shows the current branch and change count.

Plugins

MacCommander includes a plugin host for context-menu actions. The built-in sample copies a SHA-256 checksum of the selected file to the clipboard.
