Welcome

MacCommander is a dual-pane file manager inspired by classic commander-style tools. Browse two folders at once, move and copy between them with F-keys, preview files, and keep your hands on the keyboard.

Choose MacCommander Help from the Help menu at any time, or press ⌘? / ⌘/ depending on your keyboard layout. Use the search field in Help to find topics quickly.

What you can do

  • Browse with two panels side by side, each with its own tabs and history
  • Copy, move, rename, delete, compress, and extract files
  • Preview images, PDFs, text, and media; edit text or hex
  • Compare folders, find duplicates, and see Git status
  • Use an embedded terminal synced to the active folder

Start with Getting Started if you are new to the app.
