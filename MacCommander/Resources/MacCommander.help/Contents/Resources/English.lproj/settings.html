Settings & Themes

Customize how MacCommander looks and behaves

Opening Settings

Press F9, choose Settings from the toolbar gear, or use the Menu button on the function key bar.

Appearance

  • System / Light / Dark — follow or override the macOS appearance
  • Classic — Norton Commander–inspired colors: dark blue backgrounds, light cyan file text, yellow column titles, and white preview text
  • Icon size — Small, Medium, or Large for list icons

Files

  • Show Hidden Files — include dotfiles in listings (⌘⇧. also toggles)
  • Confirm Delete — ask before moving items to Trash
  • Confirm Move — ask before moving (conflicts always confirm)
  • Confirm Overwrite — default overwrite policy when transferring
  • Directories First — keep folders above files when sorting

Defaults

  • Default sort column and ascending/descending order for new panels
  • Startup folder for the left panel when the app launches

About & Help

  • MacCommander → About MacCommander — version and app icon
  • Help → MacCommander Help — this documentation in Help Viewer
