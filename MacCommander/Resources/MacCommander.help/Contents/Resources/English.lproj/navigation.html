Panels, Tabs & Navigation

Browse two places at once

Dual panes

The left and right panels are independent. Each has its own current folder, sort order, selection, and tabs. Drag the split to resize them.

Tabs

  • ⌘T — open a new tab in the active panel
  • ⌘W — close the current tab (when more than one is open)
  • ⌃Tab / ⌃⇧Tab — next / previous tab
  • Each tab keeps its own path and back/forward history

Sorting

Click a column header (Name, Size, Modified, Type) to sort. Click again to reverse the order. In Settings you can choose the default sort and whether directories appear first.

Filtering

Use the toolbar search field (Filter files) to narrow the list by name in the current view. This is separate from type-ahead quick search.

Bookmarks & volumes

  • Sidebar Favorites stores bookmarks (Home, Desktop, Documents, Downloads by default)
  • Use the toolbar bookmark button to add the current folder
  • Volumes lists mounted disks; click one to open it in the active panel

Directory watching

When files change on disk in the folder you are viewing, MacCommander reloads the list automatically after a short delay.
