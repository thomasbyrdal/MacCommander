File Operations

Work between panels and with the Finder

Copy & move

Select items in the active panel, then press F5 (Copy) or F6 (Move) to send them to the other panel’s current folder.

  • Confirm Move (Settings) — when on, Move shows a confirmation dialog
  • When off, Move runs immediately unless a same-named item already exists at the destination
  • On conflict, choose Overwrite, Skip, or Rename

Drag and drop

  • Drag between the two panes to move files
  • Drop from Finder (or another app) to copy into the target panel
  • A dialog appears only when there is a name conflict
  • Dragging one item from a multi-selection moves all selected items

Delete

  • F8 or ⌘⌫ — move to Trash (confirmation depends on Settings)
  • ⌥F8 — permanently delete

Create & rename

  • F7 / ⌘N — new folder
  • ⌘⇧N — new empty file
  • Rename from the context menu or after focusing an item
  • ⌘⇧R — batch rename with tokens such as {name}, {ext}, {counter}, {date}

Clipboard

⌘C / ⌘X / ⌘V copy, cut, and paste within MacCommander and exchange file URLs with other apps via the pasteboard.

Compress & extract

  • Compress selected items to a zip archive in the current folder
  • Extract a focused .zip archive into the current folder
