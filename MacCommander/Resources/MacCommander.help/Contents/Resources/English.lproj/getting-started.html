Getting Started

The main window at a glance

Main window

When MacCommander opens you see:

  • Favorites sidebar — bookmarks and mounted volumes
  • Left and right panels — each shows one folder
  • Status bar — file counts, selection size, free space, Git branch
  • Function key bar — F3–F10 actions (View, Edit, Copy, Move, and more)

Active panel

One panel is active at a time (highlighted border). File operations such as Copy and Move use the active panel as the source and the other panel as the destination.

Press Tab to switch the active panel, or click a panel to activate it.

Opening a folder

  • Double-click a folder (or select it and press Return)
  • Edit the path field at the top of a panel and press Return (supports ~)
  • Click a favorite or volume in the sidebar
  • Use toolbar Back / Forward, or ⌘↑ for the parent folder

Selecting files

  • Click a file to focus it
  • Space or Insert toggles selection on the focused item
  • ⇧-click selects a range; ⌘-click adds or removes items
Type a few characters while a panel is focused to jump to matching names (quick search). Press Esc to clear the quick-search query or the current selection.
